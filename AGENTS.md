# AI Usage in This Project

## Tools Used

- ChatGPT (for architecture & debugging)
- Copilot (for autocomplete)

## How AI is used

- Generate boilerplate (NestJS modules, DTOs)
- Debug errors (TypeORM, CORS, etc)
- Improve code structure

## Constraints

- No direct copy-paste without review
- All AI-generated code is manually validated
- Database logic is always double-checked

## Responsibilities split

- AI: scaffolding & suggestions
- Me: logic, structure, decision-making

## Example Prompts

- "How to handle JWT auth in NestJS?"
- "Fix TypeORM relation issue"
